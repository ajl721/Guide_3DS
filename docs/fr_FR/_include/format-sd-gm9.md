1. Appuyez sur (HOME) pour faire apparaître le menu d’action
2. Select "More..."
3. Sélectionnez "SD format menu"
4. Sélectionnez "No EmuNAND"
5. Sélectionnez "Auto"
6. Appuyez sur (A) pour accepter l’étiquette `GM9SD`
   - Optionally, you may input a custom name for the SD card
7. Lorsque cela vous est demandé, entrez la combinaison de boutons donnée pour confirmer
